Run command: 

While on root folder type in terminal: 
<br>
<i> <b> mvn compile exec:java -Dexec.args="//path to file// //password//" </b> </i>

<br>
For example:
<br>
<i><b> mvn compile exec:java -Dexec.args="C:/file.zip test123" </b></i>
